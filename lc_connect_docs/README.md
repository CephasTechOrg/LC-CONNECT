# LC Connect MVP Documentation

This folder contains the MVP documentation for the Livingstone College campus connection app.

## Product working name

**LC Connect**  
Alternative names: **BlueBridge**, **CampusBridge**, **Livingstone Connect**

## Core decision

Build a **real mobile app first** using:

- **Mobile frontend:** React Native + Expo
- **Backend API:** FastAPI
- **Database:** PostgreSQL
- **Future admin dashboard:** Next.js or React

## Documentation files

## MVP summary

LC Connect is a student-only mobile platform for Livingstone College students to safely find friends, study partners, language exchange partners, campus activities, and open connections through profiles, matching cards, mutual connections, basic messaging, and an activity board.

The app should **not** feel like a dating app first. It should feel like a safe campus connection platform that helps students break social barriers.
